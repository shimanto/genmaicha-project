# 実際の取扱商品ラインナップ(現行 + 拡張候補)

> 本ドキュメントは **「実工場で現在生産している商品」と「将来の拡張候補」** を整理した原典です。
> 本サイト上の他の資料(LP / ブランドガイドライン / 事業計画 / 原価表 / 施策)に横串で参照されます。

## 1. 現在の取扱商品(2 種)

実工場で現在稼働・出荷している商品は以下の **2 種類のみ** です。

### ① 玄米あられ(炒り玄米)

- 形状: 米粒の形を保った、クランチ系の炒り仕上げ
- 食感: カリッと軽い噛み応え、香ばしさ重視
- 用途: 茶葉(玄米茶ブレンド用)/ スナック / 菓子原料
- 焙煎特性: 中〜やや深めの釜温で、粒が割れない範囲で焙く

### ② 白くはじけた玄米(ポップライス・花)

- 形状: ポップコーン状に白く膨らんだ玄米(「花」とも呼称)
- 食感: サクッと軽く、見た目の華やかさが特徴
- 用途: 玄米茶のブレンド材(視覚的アクセント)/ 茶席用 / 可食装飾
- 焙煎特性: 高温の加圧釜で一気にはじけさせる独立工程

> **意義**: この 2 種は老舗工場 70 年の核。玄米茶の主要原料として製茶問屋に供給し続けてきた実績品。「焙 HOU / 玄米茶ちゃん」ブランドで販売する完成品も、この 2 種(またはこれらのブレンド)を土台に組み立てる。

## 2. 将来追加候補(3 種)+ 工場増設

余裕が出てきた段階で、以下の商品追加と設備投資を段階的に検討する。

### ③ 黒玄米(黒焙煎)

- 深焙煎を極限まで進め、玄米を黒色化させた高付加価値品
- 狙い: 深煎り単体飲用 / ラテ用 / 海外(北米・欧州)の「darker roast」嗜好層
- 課題: 焙煎中の苦み・エグみ管理、釜温コントロールの難易度高
- 設備: 現行のあられ用釜を延長運転で試作可 → 量産時は専用釜が望ましい

### ④ 普通炒り玄米(未発芽)

- 発芽工程を経ない、通常玄米を素直に炒ったシンプル品
- 狙い: 料理用(玄米リゾット・玄米雑炊)/ 業務用卸 / 大容量ロット
- 課題: 発芽玄米と比べた差別化ストーリーの必要性
- 設備: 既存ラインで対応可(玄米あられと同じ工程で水分調整のみ変更)

### ⑤ 発芽玄米の焙煎

- 発芽処理済み玄米を焙煎したプレミアム商品
- 狙い: GABA 含有訴求(健康志向)/ 高単価ギフト / 海外
- 課題: 発芽工程(12〜24時間の水浸 + 保温)で新設備が必須
- 設備: **発芽タンク + 乾燥機 + 焙煎機の別ライン**(工場増設の主因)

### 工場増設の検討ポイント

| 項目 | 現状 | 増設後 |
|---|---|---|
| 焙煎ライン | 1 本(あられ / ポップライス 兼用) | 2〜3 本(深焙煎専用 + 発芽専用追加) |
| 発芽設備 | なし | タンク + 温度管理棟 新設 |
| 敷地 | 現工場(宮城) | 隣接地 / 分工場 検討 |
| 初期投資 | ¥0 | 試算 ¥500万〜¥2,000万(設備次第) |
| 実施時期 | ー | 売上 ¥3,000 万を越えた頃が目安 |

## 3. ブランドサイト資料との関係性

当サイト(LP / Products / Shop / BizStory)では便宜上「朝の焙 ASA-HOU(浅焙煎)」「夜の焙 YORU-HOU(深焙煎)」などの**焙煎度を軸にした仮称 SKU**を多数掲載しています。これらは、

- 実商品(玄米あられ / ポップライス / 将来の黒焙煎など)を**焙煎度プロファイルで再編成した販売用の呼称**であり、
- 実発売時には **「玄米あられ 浅焙煎」「ポップライス花 中焙煎」** のように実体と紐づいた表示に置換します。

サイト上の「30+ SKU サンプル」や「原価表」も、実商品 2 種を起点に焙煎度・容量・パッケージで展開したバリエーション想定です。

## 4. 商品開発の基本方針

1. **既存の 2 種を最大限活かす**: いきなり新設備に投資せず、あられ / ポップライスの焙煎度・粒度・ブレンド比で差別化
2. **軌道に乗ってから拡張**: 売上 ¥1,500〜¥3,000 万が見えた段階で ③④ を追加、¥5,000 万超で ⑤ と工場増設を検討
3. **工場は 1 本のまま**を当面の前提: 新商品は極力既存釜で試作 → 量産時に増設判断
4. **地域連携で設備費を圧縮**: 商工会議所・地域金融機関・宮城県の補助金ラインを活用

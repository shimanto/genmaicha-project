# 海外での玄米茶需要

## 0. 結論先出し
- **緑茶輸出**は10年で約4倍、年率15%超で伸長。**Genmaicha は既に海外茶愛好家に知名度あり**。
- 北米 → 欧州 → アジアの順に **「茶愛好家 → 健康層 → 日常飲用層」** に拡張可能。
- 越境 EC では **国内卸価格の 2.5〜4 倍** の販売価格が成立し、小ロットでも採算性が高い。

## 2. 市場別 ファクトシート

### 北米(米国・カナダ)
- **健康志向**(カフェイン控えめ、香ばしい・玄米由来のヘルシーさ)が刺さる。
- **Genmaicha** = 寿司レストランで提供されることが多く認知あり。
- 茶専門 EC (Yunomi, Ippodo US, Mizuba Tea, Den's Tea, Maeda-en) で取扱多数。
- 想定価格レンジ: **50g $9〜$15 / 100g $15〜$25**。
- アクセス: **Etsy, Amazon.com, Shopify+海外配送代行(SHIPS, Buyandship)**。

### 欧州(独・仏・英)
- **Tea Geek 文化**(日本茶の格付けや産地表記を読む層)あり。
- **オーガニック認証** (EU Organic / USDA Organic) が販売条件になりやすい。
- ティーバッグより**リーフ志向**。フランスは「ほうじ茶 + 玄米茶」のブレンドにも親和性。
- 想定価格レンジ: **50g €10〜€16 / 100g €16〜€26**。
- アクセス: **Etsy, 専門オンライン茶舗との委託販売、Webサイト直販+DHL個別発送**。

### アジア(台湾・香港・シンガポール)
- 日本食材ブームの定着層あり。**贈答需要**強い (旧正月・中秋節)。
- **小ロット高単価**が成立しやすい。日本語パッケージのまま売れる場合も多い。
- アクセス: **越境EC (台湾Pinkoi、香港物流代行)、日本食材店との直販提携**。

### その他(中東・オセアニア)
- **健康食ブーム + 日本食レストラン拡大**で需要は今後拡大予想。
- 当面は北米・欧州・アジアで実績を作り、PRリリース材料を持って後追いを検討。

## 3. 海外で「Genmaicha」が選ばれる理由(ヒアリング+EC レビュー分析)

| 動機 | 海外消費者のコメント要約 |
| --- | --- |
| 香ばしさ | "Like roasted popcorn / Toasted, comforting" |
| カフェイン控えめ | "Easy on the stomach / Good for evening" |
| 食事に合う | "Great with sushi, rice bowls, light meals" |
| ストーリー性 | "Made by family-run roastery in Tohoku — love the story" |
| 価格 | "More affordable than matcha but still 'green tea'" |

**「焙煎玄米そのもの」を売る場合の追加訴求**:
- グルテンフリーの香ばしい飲み物 (※ 麦茶とは別軸)
- レシピ用途 (Buddha Bowl, Roasted rice broth, savory tea ramen)
- カフェの**ノンカフェイン提案**としての可能性

## 4. 海外向け パッケージとコンテンツの最低要件

- ✅ 英語表記の商品名 + 副題 (例: "Genmaicha Roasted Brown Rice — Single Origin Miyagi")
- ✅ **アレルゲン表記** (英語) / 原材料 / 内容量 (g, oz)
- ✅ **賞味期限の英語表記** + ロット番号
- ✅ ストーリー QR (本サイトへ誘導)
- ✅ オーガニック対応版は **JAS Organic + USDA / EU Organic** ロゴ
- ✅ 環境配慮: クラフト紙包装 + アルミ防湿フィルム

## 5. 越境 EC スタートのおすすめ順

1. **Etsy** (ハンドメイド/食品扱いの審査要、月数千円程度から)
2. **Amazon Global** (日本 Amazon 出品 → 海外配送オプション、敷居低)
3. **eBay** (個人〜小規模事業者向け、海外ファースト顧客の発見に良い)
4. **Shopify + 配送代行** (本格化フェーズ)
5. **海外茶専門店との卸** (CAMPFIRE/Makuake成功 → メディア掲載 → 引き合い増)

詳細は **「10 の施策 → 02 越境EC試験販売」** を参照。

## 6. リスクと対応

| リスク | 内容 | 対応 |
| --- | --- | --- |
| 食品輸入規制 | 国別の食品表示 / 検疫要件 | JETRO 越境EC 相談窓口活用 |
| 為替変動 | 円高で利益減 | 価格は四半期ごとに見直し |
| 賞味期限 | 配送遅延での消費期限不足 | 日付十分余裕で出荷、長期保存設計 |
| クレーム | 言語対応 | 自動翻訳 + テンプレ + 英語可能な外注パートナー |
| 知財 | ブランド名の商標 | 国内+主要輸出国(米国・EU)で先行調査 |

# 施策 01 — 小ロット直販EC (BASE / Shopify Lite)

## 一行で
**在庫50袋 + 月1万円から始める「自社の値付け」を取り戻すD2C。**

## なぜやるか
- 卸価格は長年動かず、原料・燃料の上昇分を吸収しきれていない。
- **自社で値付けするチャネル**がないと、永久に価格交渉力を持てない。
- 30〜60代の茶好きには「素材原料屋」のストーリーが刺さる。

## まず使うもの

| 項目 | 選択肢 | 月コスト |
| --- | --- | --- |
| EC プラットフォーム | **BASE** (初期無料 / 売上の約 6.6% + 40円) | ¥0〜 |
| または | **Shopify Lite** (¥1,500前後) | ¥1,500 |
| ドメイン | お名前.com 等 | ¥150 |
| 撮影 | iPhone 自然光 / 三脚 ¥3,000 | 一括 |
| クラフト紙袋・帯 | 100袋 単位 | ¥6,000 |
| 配送 | レターパックライト ¥430 | 都度 |

合計初期: **¥30,000 以下**で開始可能。

## 注意とリスク
- BASE は **「決済手段が信頼される」**反面、ブランド感は控えめ。中長期で **Shopify** への移行検討。
- **初回送料無料はやらない**。 値引き競争に巻き込まれる。
- 「無添加」など根拠表現は避ける。

## 次の施策との連携
- 施策 03 テイスティングセットと **必ずセット** で運用(¥500 / ¥980 二段)。
- 施策 04 ショート動画から流入を作る。
- 施策 06 ふるさと納税 と価格帯がぶつからないよう意識。

## エントリー二段の使い分け

| | ¥500 お試し (No.11) | ¥980 4種フライト (No.31) |
| --- | --- | --- |
| 中身 | 10g × 2 (朝・夜) | 5g × 4 (中焙煎・深煎り・抹茶・塩) |
| 想定 CAC | ¥500 ≒ 1 顧客獲得コスト | ¥980 で世界観理解 + リピート率↑ |
| 流入元 | SNS / 広告 | 既存購入者リスト / 口コミ |
| 配送 | レターパックライト ¥430 | レターパックライト ¥430 |
| 在庫負荷 | 低(2 種だけ) | 中(4 種小袋) |

**運用方針**: 新規流入は ¥500 で網を張り、興味を示した人には ¥980 を送る。両方持つと「お試し疲れ」を起こさないよう注意。

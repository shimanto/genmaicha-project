# 施策 01 — 小ロット直販EC (BASE / Shopify Lite)

## 一行で
**在庫50袋 + 月1万円から始める「自社の値付け」を取り戻すD2C。**

## なぜやるか
- 卸価格は長年動かず、原料・燃料の上昇分を吸収しきれていない。
- **自社で値付けするチャネル**がないと、永久に価格交渉力を持てない。
- 30〜60代の茶好きには「素材原料屋」のストーリーが刺さる。

## まず使うもの

| 項目 | 選択肢 | 月コスト |
| --- | --- | --- |
| EC プラットフォーム | **BASE** (初期無料 / 売上の約 6.6% + 40円) | ¥0〜 |
| または | **Shopify Lite** (¥1,500前後) | ¥1,500 |
| ドメイン | お名前.com 等 | ¥150 |
| 撮影 | iPhone 自然光 / 三脚 ¥3,000 | 一括 |
| クラフト紙袋・帯 | 100袋 単位 | ¥6,000 |
| 配送 | レターパックライト ¥430 | 都度 |

合計初期: **¥30,000 以下**で開始可能。

## 90日アクション

### Day 1〜10
- [ ] BASE アカウント開設 / 屋号「紡 TSUMUGI」で登録
- [ ] 最低 4 SKU(中焙煎50g/100g、深煎り50g/100g)を登録
- [ ] iPhone で商品撮影 (背景は和紙 / 自然光)
- [ ] 商品説明文 600字 x 4本

### Day 11〜30
- [ ] Instagram アカウント開設 → 投稿×10
- [ ] 親類・知人に **告知 DM**(値引きなし、ストーリー共有)
- [ ] 初回30個の出荷 → **同梱手書きメモ**

### Day 31〜90
- [ ] **テイスティングセット** (¥980)を追加
- [ ] レビュー依頼 → 5★レビュー 5件以上を集める
- [ ] **メルマガ / LINE 公式** 1,000円以上購入者に送付
- [ ] **3ヶ月目に第1回SALE**(春の新茶連動など)

## 注意とリスク
- BASE は **「決済手段が信頼される」**反面、ブランド感は控えめ。中長期で **Shopify** への移行検討。
- **初回送料無料はやらない**。 値引き競争に巻き込まれる。
- 「無添加」など根拠表現は避ける。

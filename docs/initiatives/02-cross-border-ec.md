# 施策 02 — 越境EC試験販売 (Etsy / Amazon Global / eBay)

## 一行で
**海外配送代行を使い、北米・欧州への販売チャネルを実需テスト。**

## なぜやるか
- 海外オンラインEC では国内卸の **2.5〜4 倍**の販売価格が成立。
- 物理的な海外進出より、まず **本サイトに英語でアクセスが来るか**を試す。
- 商品を 1 つでも売って **「越境売上の実績」** を作ると、補助金・融資・取材機会が増える。

## まず使うもの

| 項目 | 選択肢 | 初期費 |
| --- | --- | --- |
| Etsy 出店 | $0.20/品 リスト + 6.5% 取引 + 3% 決済 | ¥3,000 |
| Amazon Global | 大口 ¥5,000/月 | ¥5,000 |
| eBay | 出品手数料(Free Listings あり) | ¥0〜 |
| 英語商品ページ | DeepL + ChatGPT 下書き → 外注最終チェック | ¥10,000 |
| 配送代行 | CUBE / Buyandship / Tenso | ¥0〜 |
| 決済受け取り | PayPal / Wise / 各PFのデフォルト | 月¥0 |

合計初期: **¥30,000〜¥50,000**程度。

## 90日アクション

### Day 1〜10
- [ ] **JETRO 越境EC 相談**を予約・実施
- [ ] **商標出願**(国内 + 米国・EU)を弁理士に依頼開始
- [ ] 英語パッケージ表示の最低要件確定 (`docs/export-checklist.md` 参照)

### Day 11〜30
- [ ] Etsy 出店 / 商品 4 SKU 登録(50g 中焙煎・深煎り、テイスティング、ギフト)
- [ ] Amazon.com 出品(海外配送)
- [ ] 配送代行 1 社契約 → サンプル発送

### Day 31〜90
- [ ] **第 1 便発送**(初注文 1〜2 件 想定) → 体験録を社内マニュアル化
- [ ] レビュー収集テンプレ送付
- [ ] 月次レビューで **CVR・注文単価・配送コスト**を可視化

## KPI 90日

| KPI | 目標 |
| --- | --- |
| 出店プラットフォーム | Etsy + Amazon (最低2) |
| 海外注文数 | 累計 5 件以上 |
| 海外売上 | 累計 ¥30,000 以上 |
| 海外レビュー | 1 件以上(★4以上) |

## 注意とリスク
- 食品の **輸入規制** は国別に異なる。FDA 登録など事前確認必須。
- 配送ロス想定 1〜3%。**保険込みの配送代行**を選ぶか、**代金引換不可**で運用。
- 為替変動で利益が削られるため、**四半期ごとに販売価格を見直し**。

# 施策 08 — お茶クリエイター / インバウンドガイドへのギフティング

## 一行で
**英語コンテンツを持つ茶YouTuber・在日インバウンド向けガイドへ無償提供で口コミを設計。**

## なぜやるか
- 海外向けに広告を打つには **言語の壁・予算の壁**が大きい。
- 既に **海外フォロワーを持つ日本茶クリエイター**へ送る方が ROI が高い。
- インバウンドガイド(訪日客向けツアー業者)経由で、**訪日客が宮城・東北を訪れた時の手土産**として認知させる。

## 想定先

| 先 | 例 | 接点 |
| --- | --- | --- |
| 海外茶 YouTuber | "Tea geek" 系の海外発信者(英語 / 仏語) | DM / メール |
| 日本茶 紹介 Instagrammer | 茶器・茶葉に詳しい日本人 | コメント → DM |
| インバウンド ガイド | 訪日客向け茶道体験 / 食ツアー業者 | 業界誌 / Web |
| 日本茶 専門店 海外 | 米国 / 欧州 茶舗 | サイト問い合わせ |
| 在京 海外特派員 | 食・ライフスタイル系 | フリーランス記者DB |

## 90日アクション

- [ ] 候補 20 名 のリスト化(SNS フォロワー 5,000 以上中心)
- [ ] **テイスティングセット 20 個 + 英語ガイド + ストーリーカード**を発送
- [ ] 1 週間後に **シェア依頼の柔らかいメッセージ**(強要しない)
- [ ] **シェアされた投稿は本サイトに集約** (ソーシャルプルーフ)

## KPI 90日

| KPI | 目標 |
| --- | --- |
| 発送数 | 20 件 |
| 公開シェア数 | 6 件以上 |
| 海外問い合わせ | 5 件以上 |
| 海外EC 流入 | 月 100 PV 以上 |

## 注意とリスク
- **「ステマ」と取られない設計**(ハッシュタグ #PR / #gifted を必ず明示)
- 大手インフルエンサー一発狙い → **失敗時のダメージ**大なので、**中規模 20 名分散**でリスク軽減
- 発送物 = 試飲規模で OK。**大量提供しない**(価値が落ちる)

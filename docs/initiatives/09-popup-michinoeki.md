# 施策 09 — 道の駅 / 地元百貨店 POPUP

## 一行で
**宮城の道の駅+JR駅ナカ百貨店 POPUPで、観光客と仙台市内の常連を同時に取りに行く。**

## なぜやるか
- 道の駅は **宮城観光客の必須立ち寄り場所**。 県外への持ち帰り需要に直撃。
- 仙台駅・S-PAL の催事 POPUP は **法人ギフト需要**を吸い上げる装置として優秀。
- 出店期間が短い分、**仕込みと体力消耗の見通し**が立てやすい。

## 想定アクション

### 道の駅 (常設委託販売)
- 県内 5 件の道の駅で **委託販売(売上の 70-80% 還元)** を試行
- 商品: 50g 中焙煎・深煎り / テイスティングセット / ティーバッグ
- 委託契約 = 現地スタッフが在庫・販売 / 当社は補充のみ

### S-PAL / さくらの百貨店 / 三越仙台 等の POPUP
- 春・秋・年末の催事に出店(共同ブースが取りやすい)
- 母娘で交代しながら **3〜5 日のスポット出店**
- 試飲提供 + その場販売 + 名刺獲得

## 90日アクション

- [ ] 道の駅 5 件 と委託販売条件を交渉
- [ ] 委託契約書作成(弁護士レビュー)
- [ ] S-PAL 等 1 件と POPUP 交渉
- [ ] **POPUP 用什器**(折りたたみ机・布・什器)を一式整備
- [ ] 1 件目 POPUP 実施 → 売上 / 名刺数を可視化

## KPI 90日

| KPI | 目標 |
| --- | --- |
| 道の駅 委託先 | 3 件 |
| 道の駅 月売上(合計) | ¥30,000 |

## 注意とリスク
- 委託販売は **回転速度**を見る(売れない道の駅は早めに撤退)
- POPUP は **準備〜搬入〜搬出の体力消費**が大きい → **製造を1日休む覚悟**で挑む
- 観光客向け = **持ち帰り重視** → 軽量・割れにくいパッケージが基本

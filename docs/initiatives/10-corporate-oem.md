# 施策 10 — 法人ノベルティ / OEM 受託

## 一行で
**商工会議所つながりの中小企業の周年・株主優待・贈答需要に対し、小ロットOEMで応える。**

## なぜやるか
- 法人ノベルティ・OEM は **少ない案件数で高単価**(1案件 ¥30,000〜¥500,000)。
- 商工会議所の経営指導員経由で **見込み顧客の紹介**を受けやすい。
- **個人D2Cと同じ製造ラインで完結**するため、追加設備投資が小さい。

## 商品メニュー(再掲 / `docs/oem-contract-template.md` 参照)

| メニュー | 想定単価 | 最低ロット |
| --- | --- | --- |
| 周年ノベルティ 8g x 100 | ¥38,000〜 | 100個 |
| 株主優待 50g x 100 | ¥120,000〜 | 100セット |
| 結婚式 引出物 50g x 2 + カード | ¥75,000〜 | 50セット |
| 法人贈答 ギフトボックス | 個別見積 | 30セット |
| ふるさと納税 専用パッケージ | 個別見積 | 100個 |

## 90日アクション

- [ ] OEM 雛形(`docs/oem-contract-template.md`)を **PDF化** して持参資料化
- [ ] 商工会議所 経営指導員に **「OEM 受託します」**を明示・ヒアリング依頼
- [ ] 出店イベントで **法人決裁者** の名刺を集める
- [ ] **3 件の見積依頼**を獲得 → 1 件の成約を目標

| 見積提出 | 3 件 |

| 成約 売上 | ¥80,000 以上 |
| リピート確率 (12ヶ月) | 30% |

## 注意とリスク
- **デザイン制作工数**を見積から外すと赤字化。**¥30,000〜の設計費**を必ず提示。
- 周年贈答は **時期集中**(秋〜年末・春)。 製造ピーク管理が必要。
- **食品PL保険**への加入を検討(法人渡しの安心材料)。
- 大企業案件の場合 **支払サイト 60 日以上**もあり、資金繰りに注意。

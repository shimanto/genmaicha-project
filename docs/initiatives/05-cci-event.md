# 施策 05 — 商工会議所 / 物産展 出店

## 一行で
**宮城県商工会議所のネットワークと補助制度を最大活用した名刺獲得装置。**

## なぜやるか
- 商工会議所は **最も低コストで信頼性のある販路接続**装置。
- 補助金・出店枠・経営指導員のサポートが体系的に整っている。
- 1 回の出店で **法人バイヤー名刺 5〜20 枚**を獲得できることもある。

## 想定アクション

### 出店候補
| イベント | 時期 | 規模 | 想定費用 |
| --- | --- | --- | --- |
| 宮城県物産展(東京・仙台) | 春・秋 | 中 | 補助込みで実費数万円 |
| 商工会議所主催 商談会 | 年 2 回 | 中 | 出展料あり / 補助対象 |
| 食品 関連 EXPO(FOODEX 出張版等) | 春 | 大 | 共同ブースなら数万円 |
| 銀行・信金主催 ビジネスマッチング | 通年 | 小〜中 | 多くは無料 |

### 必要準備物
- [ ] 試飲セット 100 セット(施策 03 配布版を活用)
- [ ] ブランド A4 リーフレット(QR で本サイト誘導)
- [ ] 名刺 200 枚 / アンケートカード
- [ ] 簡易レジ(Square 等のモバイル決済)
- [ ] 服装(統一感のあるユニフォーム不要、清潔感重視)

## 90日アクション

- [ ] 商工会議所 経営指導員 訪問・登録
- [ ] 出店候補 3 件をリストアップ + エントリー
- [ ] 1 件目の出店実施 → **来場者DB**(名前 / 所属 / メール)を構築
- [ ] フォローアップメール(出店後 2 週間以内)

## KPI 90日

| KPI | 目標 |
| --- | --- |
| 出店件数 | 1〜2件 |
| 名刺獲得 | 30 枚 |
| 試飲提供 | 200 杯 |
| その場販売 | ¥50,000 |
| 後追い商談 | 3 件 |

## 注意とリスク
- 出店時の **食品衛生** 取扱許可が必要なケースあり (保健所事前確認)
- 母娘 2 人で出店すると工場が止まる → **代替シフト** or **出店日は出荷オフ**
- リーフレット配布は **「QR で本サイト」** に統一(印刷物の量を減らす)

# 施策 03 — 焙煎玄米 2種テイスティングセット

## 一行で
**中焙煎×深煎り 5g×4袋の試飲セットで、ブランド理解を低価格で提供。**

## なぜやるか
- 焙煎玄米単体の購入は **「飲み方が分からない」「香りが想像できない」** が主な障壁。
- 5g x 4袋 (1〜2杯ずつ試せる) で **¥980** なら障壁が下がる。
- 試飲セット購入者は **本商品の購入率が3〜5倍**(他カテゴリの一般傾向)。

## 商品仕様

| 項目 | 内容 |
| --- | --- |
| 中身 | 5g x 4袋(中焙煎 / 深煎り / 抹茶混合 / 塩玄米) |
| パッケージ | クラフト紙箱 + 透明窓 + 帯紙 |
| 同梱物 | 飲み方ガイド(紙片 1 枚) |
| 価格 | **¥980 (税込・送料込)** |
| 対象チャネル | 自社EC + イベント + ふるさと納税 |

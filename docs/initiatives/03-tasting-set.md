# 施策 03 — 焙煎玄米 テイスティングセット(¥500 / ¥980 二段構成)

## 一行で
**「とりあえず一杯」¥500 と「世界観全部」¥980 の二段で、入口の摩擦を最小化。**

## なぜやるか
- 焙煎玄米単体の購入は **「飲み方が分からない」「香りが想像できない」** が主な障壁。
- 価格・量ともに低くした **¥500 (10g×2)** で「迷わず買える」入口。
- そこから一歩進みたい人向けに **¥980 (5g×4)** で世界観全体を体験。
- 試飲セット購入者は **本商品の購入率が3〜5倍**(他カテゴリの一般傾向)。

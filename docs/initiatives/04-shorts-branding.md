# 施策 04 — 1日10分のショート動画ブランディング

## 一行で
**釜の音・湯気・玄米のはじけ。母娘の手元を撮るだけで成立する素材庫。**

## なぜやるか
- 焙煎風景は **被写体としての強度が極めて高い** (音・色・温度が伝わる)。
- 顔出し不要・台本不要・編集ほぼ不要で **1日10分のオペレーション**にできる。
- Instagram Reels / TikTok / YouTube Shorts の **3 媒体に同じ素材を流用**可能。

## オペレーション

| 時間 | 作業 |
| --- | --- |
| 朝 8:00 | iPhone を釜の前に固定 / 焙煎中の音と湯気を 60 秒撮影 |
| 朝 8:05 | 1 本選んで CapCut で字幕 1 行追加 |
| 朝 8:10 | 投稿(Reels に同時投稿で TikTok / YouTube Shorts へクロスポスト) |

→ **1日10分 / 月20本** の更新を目指す。

## コンテンツ枠

| 枠 | 例 | 月本数 |
| --- | --- | --- |
| 焙煎風景 | 玄米のはじける音 / 湯気 | 8本 |
| 淹れ方 | 中焙煎・深煎りの淹れ方比較 | 4本 |
| 母娘の手元 | 包装・選別 / シーラー音 | 4本 |
| 海外向け 英語字幕 | 同素材を再利用 | 4本 |

## 90日アクション

- [ ] iPhone 三脚 + 外部マイク (¥5,000以下) 購入
- [ ] CapCut テンプレ作成(タイトル・字幕の固定)
- [ ] 月 20 本ペースで継続
- [ ] **3ヶ月で「最も伸びた 3本」**を分析し、商品ページから埋め込む

## KPI 90日

| KPI | 目標 |
| --- | --- |
| 投稿本数 | 60本 |
| Instagram フォロワー | 500人 |
| 投稿あたり再生数 平均 | 1,000 回 |
| EC 流入数 | 月 100 PV |

## 注意とリスク
- **顔出ししない**を原則に(母娘の安全 + 撮影の心理的ハードル下げる)
- **音楽は商用ライセンス込みのもの**(プラットフォーム標準音源優先)
- **撮影時間=製造を止めない時間** に限定(あくまで副次的活動)

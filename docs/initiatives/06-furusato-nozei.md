# 施策 06 — ふるさと納税 返礼品登録

## 一行で
**宮城県内自治体への返礼品提供で、無広告・在庫リスク低めの売上を立てる。**

## なぜやるか
- ふるさと納税は **広告費ゼロで全国の高所得層にリーチ**できる稀有なチャネル。
- 自治体ポータル(さとふる / 楽天 / ふるなび)で **検索流入**が常時ある。
- 寄付額の 30% 程度が事業者側に入る形が標準で、**粗利率も悪くない**。

## 想定建付け

| 項目 | 内容 |
| --- | --- |
| 登録自治体 | 宮城県内の本拠地自治体 + 隣接 1〜2 自治体 |
| 返礼品 | テイスティングセット / 50g x 2袋 / ギフトボックス 等 |
| 寄付額帯 | ¥5,000 / ¥10,000 / ¥20,000 |
| ポータル | さとふる + 楽天ふるさと納税(まず2つ) |

## 90日アクション

### Day 1〜30
- [ ] 商工会議所 経営指導員に **ふるさと納税担当**へ繋いでもらう
- [ ] 自治体担当者と面談 → 返礼品要綱・地場産品基準の確認
- [ ] パッケージに **自治体名/原産地表示**を追加

### Day 31〜60
- [ ] 申請書類提出 → 採択
- [ ] ポータル掲載素材(写真・説明文)作成
- [ ] 包装の専用版(自治体ロゴ入り)を試作

### Day 61〜90
- [ ] さとふる + 楽天で同時公開
- [ ] **年末駆け込み需要** に間に合わせるため遅くとも 11 月公開
- [ ] 月次オーダーレポートを記録

## KPI 90日

| KPI | 目標 |
| --- | --- |
| ポータル登録数 | 2 ポータル |
| 返礼品 SKU 数 | 3 SKU |
| 90日 寄付件数 | 30 件以上 |
| 売上(自治体支払額) | ¥150,000 以上 |

## 注意とリスク
- **地場産品基準**(原材料・加工地比率)を満たすこと
- 年末集中による **製造ピーク** の体力配分
- 自治体側の事務スピードに依存(着手 → 公開 まで 60〜90日)

# 商品開発 ロードマップ

> 既存事業 (玄米茶用 焙煎玄米の卸) を **止めずに**、新ブランド「紡 TSUMUGI」の商品ラインを段階展開するための計画。

---

## 0. 開発原則

1. **既存設備でできる範囲から始める**(初期投資最小)
2. **母娘 2人 で製造完結する SKU**を優先
3. **国内 D2C → 越境 EC → 法人 OEM の順**にチャネル拡張
4. **ストーリーが語れる SKU だけ出す**(雑多な品揃えはしない)
5. **賞味期限管理が現実的な内容量**(50g / 100g 中心)

## 5. SKU 開発判断のチェックリスト

新 SKU を出すか迷った時の判断基準:

- [ ] 母娘 2 人で 1 ロット製造完結できるか？
- [ ] 賞味期限 6ヶ月以上を確保できるか？
- [ ] 平均粗利率 50% 以上を確保できるか？
- [ ] 既存ブランドガイドラインで語れるストーリーがあるか？
- [ ] 既存 SKU と被って共食いしないか？
- [ ] 補助金 / 商工会議所支援との整合性があるか？

5 つ以上 ✅ なら GO。それ以下は次フェーズ送り。

---

## 6. 製造 - 在庫 - 廃棄の管理

- 月次で **在庫棚卸**(母娘 + 商工会議所推薦の経営指導員レビュー)
- 賞味期限残 3ヶ月切ったら **早割セール枠**で消化
- ロット番号を **印字**(食品事故時のトレース確保)
- 廃棄ロスは **3% 以下**を目標

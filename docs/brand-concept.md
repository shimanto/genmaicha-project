# 焙 HOU ブランドコンセプト

## ブランドステートメント

**「焙煎玄米だけで、玄米茶の常識を焙き直す。」**

宮城県の小さな焙煎工場から、若き女性経営者と母が立ち上げる、焙煎玄米専業の独立ブランド。
70年近くにわたり「玄米茶の玄米だけ」を焙ってきた家業を、
ブレンド素材から"単品で選ばれる主役"へと再定義します。

## ブランドストーリー(要約)

- 祖父母が宮城県で焙煎玄米工場を創業
- 母がそれを引き継ぎ、30年以上にわたって焙煎を続けてきた
- 20代の娘(穂香)が事業継承と並行し、新ブランド「焙 HOU」を立ち上げる
- 国内の嗜好変化(ノンカフェイン・単一素材回帰)と海外の日本茶需要の拡大を追い風に、
  BtoB 卸中心モデルから BtoC/D2C/海外直販 へと段階的にシフト

## ブランドバリュー

| バリュー | 内容 |
|---------|------|
| 素材主義 | 焙煎玄米そのものを主役に据える。ブレンドは"用途の一つ"として扱う |
| 継承 | 家族経営・事業承継・地域コミュニティとの関係を資産として捉える |
| 顔の見える発信 | 広告費より"人肌の物語"。経営者本人がブランドの顔になる |
| 小さく始める | 資金・人員・生産能力を無理に広げず、小ロットでの直販から積み上げる |

## トーン&マナー

- 色: 焙煎玄米の焦げ茶・琥珀・新芽の緑。白背景に温かい差し色を置く
- 書体: 明朝系(Noto Serif JP)を主、ゴシック(Noto Sans JP)を従
- 写真: 工場の温度感・手仕事・湯気・粒感を重視
- 文体: 「わたし/うちは…」語り、過度な装飾をしない平易な日本語

## 禁忌事項

- 祖父母・母・取引先の個人情報を本人同意なしに発信しない

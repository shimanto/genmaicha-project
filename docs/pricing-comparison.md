# 国内 vs 海外 価格比較

## 0. 結論
- **同等内容量**で見ると、海外オンライン小売は国内卸の **2.5 〜 4 倍** の価格帯が普通。
- 越境EC では送料・手数料が重いが、**販売価格自体が高い**ため粗利は国内D2Cと同等以上を確保可能。
- 当社にとっての勝ち筋は「**焙煎玄米単品 50g/100g** を **クラフト・シングルオリジン**として位置付ける」こと。

## 5. 価格を守るためのルール

1. **値引き販売は1年に2回** (春・秋のセールのみ)。常時セール禁止。
2. **送料無料ライン** は ¥3,500 / $35 以上の購入時のみ。
3. **OEM 受託は最低 50 個から**。それ未満は規模感が合わないので断る。
4. **クラウドファンディング** は早割を含めても **定価の 80% 以上**を維持。
5. **海外価格** は四半期ごとに為替+運賃で見直し。

## 6. リスクと注意

| リスク | 内容 | 対応 |
| --- | --- | --- |
| 価格競争 | 大手 / 廉価ブランドが参入 | 「焙煎玄米専業」「ストーリー」で離す |
| 為替 | 急速な円高で利益減 | EC側の販売価格を四半期で更新 |
| 送料高騰 | 国際配送料アップ | 100g 以上 SKU を主力に / 卸チャネル拡張 |
| プラットフォーム改定 | Etsy / Amazon の手数料変更 | 自社 Shopify への少しずつの移行 |

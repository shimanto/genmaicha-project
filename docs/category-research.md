# 玄米茶 カテゴリー調査

## 0. このページの目的
玄米茶という単語の中に、どのような商品ジャンルが含まれているかを整理し、**「紡」が攻めるべき位置**を明確化します。
公開データとレンジ推定をベースにした構想資料です。

---

## 1. 玄米茶とは何か(構造分解)

玄米茶は **「茶葉 + 焙煎玄米」** を主構成要素とする日本茶カテゴリーです。
構成要素ごとに「ばらつき」が大きく、同じ「玄米茶」と書いてあっても中身は大きく異なります。

| レイヤ | 主な選択肢 | バリエーション源 |
| --- | --- | --- |
| 茶葉ベース | 番茶 / 煎茶 / 茎茶 / 抹茶 | 価格帯・産地で大差 |
| 玄米 | 焙煎玄米(中焙煎) / 深煎り / 黒玄米 / 雑穀混合 | **当社の主戦場** |
| ブレンド比率 | 茶葉 : 玄米 = 5:5 / 7:3 / 9:1 など | メーカーが独自設計 |
| 仕上 | 抹茶入り玄米茶 / ほうじ茶ブレンド | 高単価帯で多い |
| 包装形態 | リーフ缶 / バルク袋 / ティーバッグ / カフェ業務用 | チャネルで分岐 |

**当社の立ち位置**: 上記のうち「玄米」レイヤ専業 (中焙煎・深煎り 2 種)。
今後はここを起点に **「焙煎玄米そのもの」を主役にした商品**を作ります。

## 3. 顧客セグメント

### A. 国内 茶飲用層
- 30〜60代 / 健康志向 / 自宅で急須を使う層
- 価格感度は中程度。**「素材原料の良さ」を語ると刺さる**。

### B. 国内 ギフト層
- 親へのお歳暮、法人贈答 (5,000円〜2万円帯)
- パッケージとストーリーが買われる。**「老舗 + 4代目女性」は強い**。

### C. 海外 茶愛好家(Tea Geek)
- 北米・欧州 / 茶専門店ECや SNS で情報収集
- **Genmaicha は既に知名度あり**、深掘り提案で単価が伸びる。

### D. 海外 日本食材ユーザー
- アジア(台湾・香港・シンガポール)+ 欧米日本食ファン
- スーパー/オンラインで日常使い。価格は中程度〜高め。

### E. 業務用(料理用)
- 焙煎玄米のお出汁、ふりかけ、玄米ご飯トッピング
- カフェ・和食店・健康レシピメディアと相性◎

# 越境EC スタート 輸出チェックリスト

> 焙煎玄米の小ロット越境ECを始めるためのToDoリスト。 母娘 2 人で **3ヶ月以内**に第1便を出すことを目標にする。

---

## A. 法令・許認可

- [ ] **食品表示法** に準拠した日本国内向け表示を確定する
- [ ] **食品衛生法**: 製造所固有記号・製造者情報の記載
- [ ] 輸出先別 食品輸入規制の確認
  - [ ] 米国: FDA Food Facility Registration / Prior Notice 提出
  - [ ] EU: 食品事業者登録 / 一部品目 Health Certificate 必要
  - [ ] 英国: ブレグジット後の独自規定確認
  - [ ] 台湾: 輸入業者経由が安心(直送には食品輸入手続き)
  - [ ] 香港: 規制比較的緩いが原産地表示必要
- [ ] 必要に応じて **JETRO 越境EC 相談窓口**で個別相談

---

## B. 商標・知財

- [ ] 国内: 「玄米茶ちゃん / GENMAICHA-chan」第30類 茶 出願
- [ ] 米国: USPTO 先行調査 + 出願検討
- [ ] EU: EUIPO 先行調査 + 出願検討
- [ ] パッケージ意匠: 国内 + 主要国
- [ ] **専門家**(弁理士)経由で進める

---

## C. パッケージ・表示

- [ ] 英語商品名 (例: "Single Origin Roasted Brown Rice — Genmaicha Style")
- [ ] 内容量(g + oz の併記)
- [ ] 原材料 / アレルゲン(英語)
- [ ] 賞味期限(YYYY-MM-DD 形式)
- [ ] 製造ロット番号
- [ ] 原産国(Made in Japan)
- [ ] 焙煎メーカー / 連絡先(英語)
- [ ] バーコード(EAN/UPC) — Etsy/Amazon 用
- [ ] ストーリー QR(本サイトへ誘導)

---

## D. 物流

- [ ] 配送代行サービスの選定
  - [ ] **CUBE / Buyandship / Tenso 等の比較**
  - [ ] 1個あたりの配送費(50g / 100g)
  - [ ] 追跡有無 / 補償有無
- [ ] **国際郵便** (EMS / 国際小包 / 国際eパケット) の使い分け
- [ ] **保管温度**:常温で問題ないが、夏期の積み置き対策
- [ ] **梱包**: クラフト紙袋 + アルミ防湿フィルム + 緩衝材

---

## E. 決済・税

- [ ] 越境EC各プラットフォームの **入金通貨・為替** 把握
- [ ] **PayPal / Wise**等のマルチ通貨口座開設
- [ ] **消費税 輸出免税**の手続き(輸出証明)
- [ ] **税理士** との顧問契約(海外売上計上方法)
- [ ] **個人事業 → 法人化** のタイミング検討(売上規模次第)

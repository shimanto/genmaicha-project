## 4. 強み・差別化(SWOT)

### Strengths
- **100年近い焙煎ノウハウ**(他社が真似しにくい時間資産)
- **焙煎玄米専業**(玄米茶大手とは違うレイヤ)
- **4代目女性経営者の事業承継ストーリー**(メディア・補助金で評価されやすい)

### Weaknesses
- 設備老朽化(更新が必要)
- マーケ・営業時間が極小
- 雇用拡大には体力的限界

### Opportunities
- 海外 Genmaicha 需要拡大
- 補助金(事業再構築・小規模持続化)枠
- 商工会議所ネットワーク

### Threats
- 大手廉価ブランドの参入
- 為替変動・送料高騰
- 災害(東北地震・ライフライン)

## 10. リスクと対策

| リスク | 内容 | 対策 |
| --- | --- | --- |
| 経営者(玄米茶ちゃん本人)の体調不良 | 1人依存 | 母 + パートタイム 1名育成 / マニュアル化 |
| 母の高齢化 | 体力低下 | 包装ライン半自動化(投資項目化済) |
| 設備故障 | 焙煎釜停止 = 売上ゼロ | 修理積立 / 増設で冗長化 |
| 取引先卸の縮小 | 既存売上減 | 新ブランドで分散 / 直販比率増 |
| 為替・物流費 | 海外利益圧迫 | 価格四半期見直し / 高単価SKU重視 |
| 風評・SNS事故 | 小規模事業の脆弱性 | 公開ガイドラインを家族と共有 |

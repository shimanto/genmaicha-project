# 焙 HOU 事業計画(概要)

## 1. 事業の背景

- 宮城県内で70年近く、焙煎玄米専業として稼働してきた家族経営の工場
- 現状は玄米茶メーカー向けのOEM卸が中心
- 母と娘の2名体制で製造・営業・発送を担っている
- 20代の娘(仮名: 玄米茶ちゃん)が事業継承と並行し、独立ブランド「焙 HOU」を立ち上げる

## 2. 市場機会

| 市場 | 方向性 |
|------|--------|
| 国内リーフ玄米茶 | 成熟市場。棚価格は安値帯と高級帯に二極化 |
| 国内ノンカフェイン | 継続成長。ほうじ茶・焙煎玄米に追い風 |
| 海外日本茶 (抹茶起点) | CAGR 7〜10%で拡大。"Genmaicha"も単語認知が進む |
| 地域×ストーリー消費 | 事業承継・女性経営者・東北というタグが評価されやすい |

## 3. プロダクト戦略

- 初期SKUは 浅焙煎 ASA-HOU / 深焙煎 YORU-HOU / お試しセット の3種
- 容量: 100g缶・100g袋・20g×2セット
- 後期拡張: 200g詰替え / 業務用500g / 缶ギフトセット / カフェB2B

## 4. チャネル戦略

| フェーズ | 期間 | 主な打ち手 |
|---------|------|-----------|
| Phase 1 | 0〜3ヶ月 | Shopify D2C立ち上げ / LINE公式 / SNS起点の物語発信 |
| Phase 2 | 3〜9ヶ月 | 国内専門茶店・カフェ卸 / 地域メディア露出 / マルシェ出店 |
| Phase 3 | 9〜18ヶ月 | 海外Shopify Markets / 海外専門茶店卸 / 展示会初参加 |
| Phase 4 | 18ヶ月〜 | 海外B2B継続取引 / 商品ライン拡張 / 生産投資 |

## 5. 体制

- 代表: 玄米茶ちゃん (20代 女性経営者 / 仮名・企画・発信・D2C運営)
- 焙煎責任: お母さん (母 / 現場統括)
- パートナー: 商工会議所・地域金融機関・デザイナー(業務委託)
- 採用計画: 売上¥1,500万を超えたタイミングでパートタイム1名

## 8. KPI

- LINE公式 友だち数 / 開封率
- D2C 月間注文数 / リピート率
- SNS フォロワー増 / 保存率
- 卸取引先数 / 継続率
- 補助金採択件数 / 融資枠獲得額

## 9. リスクと対処

- 工場能力(母娘2名)の上限 → 受注ベースの小ロット焙煎で過剰生産を防ぐ
- 既存OEM取引の棄損 → 既存先とは別SKU・別価格帯で棲み分けを明文化
- 資金繰り → 補助金は交付前払いのため、つなぎ資金を公庫で確保
- 本人の発信疲れ → 撮影・投稿のテンプレ化と月次レビューで負担を削減

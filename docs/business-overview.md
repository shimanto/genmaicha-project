# 事業説明 — 玄米茶ちゃん

## 1. わたしたちのいま

宮城県内の小さな焙煎工場で、**「玄米茶用の焙煎玄米」**だけを作り続けている老舗です。
祖父祖母の代から数えて 4 代目「玄米茶ちゃん」(20代女性経営者、ニックネーム)が、母と二人で 1 釜を回しています。

| 項目 | 内容 |
| --- | --- |
| 拠点 | 宮城県内 焙煎工場 1棟 |
| 体制 | 母娘 2名 (他従業員なし) |
| 設備 | 直火焙煎釜 1基 + 選別・包装ライン |
| 主商品 | 焙煎玄米(中焙煎・深煎り の 2種) |
| 主販路 | 茶問屋・地場ブレンダーへの卸 |
| 取引年数 | 30年以上の継続取引先複数 |

製造しているのは **「茶葉とブレンドする前段階の焙煎玄米」**。
つまり、消費者の手元に届く玄米茶の「玄米部分」を担っている、いわば素材メーカーです。

---

## 2. 抱えている 4 つの課題

### 課題 ① コスト捻出が難しい
卸価格は長年大きく変わらず、原材料費・燃料費・包装資材の上昇分を吸収しきれていません。
新しい商品開発に回す**「自由に使えるお金」**がほぼゼロの状態です。

### 課題 ② 時間がない
日々の焙煎・包装・出荷で、母娘の手はほぼ常に塞がっています。
**マーケティングや商品開発に「まとまった半日」を取るのが難しい**のが現実です。

### 課題 ③ 雇用を増やしにくい
雇用すればコストが増え、教育コストも自分達の時間を奪います。
最初の一歩は、雇用に頼らない「自分達と少しの外注」で完結する施策が必要です。

### 課題 ④ ブランドが「裏方」のまま
玄米茶の「玄米部分」だけを供給しているため、消費者からの認知はゼロに近い状態です。
事業承継のタイミングで **自社ブランド「玄米茶ちゃん」を立ち上げ、表に立つ**準備をしています。

## 3. 「玄米茶ちゃん」が解こうとしていること

### A. 既存事業を**止めずに**新規ブランドを並走させる
卸先との関係は維持。新ブランド「玄米茶ちゃん」は、釜が空く時間帯と休日を活用してスモールスタートします。

### B. 「焙煎玄米そのもの」を主役にする
玄米茶は通常「茶葉が主、玄米が従」。
わたしたちは**焙煎玄米単体での飲用・料理利用・贈答利用**を提案し、新しい使い方を作ります。

### C. 国内 D2C と海外越境 EC の二刀流
- 国内: ふるさと納税 + 自社EC + 商工会議所ネットワーク
- 海外: Etsy / Amazon Global / 越境EC(発送代行使用)で **小ロット試験販売**

### D. 小売カテゴリーへの段階拡張
将来的には **ラテ原料 / RTD ボトル飲料 / 菓子 / 料理用粉砕パウダー / ソフトクリーム原料** など、
焙煎玄米を素材として複数カテゴリに展開します(`docs/product-roadmap.md` 参照)。

### E. 「事業承継 × 新規事業」というストーリーを共有する
本サイトと漫画は、**商工会議所・金融機関・補助金審査者**に対し、
構想と実行プランを 1 リンクで共有することを目的に設計しています。
